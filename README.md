## Run locally

Open `index.html` in a browser.

## Deploy as a website

This app is static and can be hosted on:

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel

Upload these files and folders:

- `index.html`
- `app.js`
- `styles.css`
- `assets/`
- `.nojekyll`

The `reference/` folder is not required for the live website.

## Discord Activity

After deploying the website to an HTTPS URL, use that URL as the Activity URL in Discord Developer Portal.
